#02 Trabajando con Pokemon
Vamos a desarrollar el programa de Python que nos permita, a partir de una lista de palabras, encontrar la lista más larga posible de palabras de manera que la última letra de una palabra coincida con la primera de la siguiente palabra. Para probar el programa, la lista de palabras con las que vamos a trabajar son las que encontramos en el fichero pokemon.txt
